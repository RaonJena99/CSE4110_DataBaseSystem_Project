Real Estate Management System

This C program serves as a basic Real Estate Management System. It connects to a MySQL database and allows users to execute predefined SQL queries for managing properties, agents, and sales transactions.

Features:

Database connectivity with MySQL.
Interactive menu for query selection.
Ability to execute batch operations using input files.
Insertion of new sales transactions and agents.
Basic error handling.
Usage:

Compile the program with a C compiler.
Configure the MySQL database connection parameters in the source code.
Prepare input files with SQL queries if needed.
Run the executable and follow the menu prompts.
Dependencies:

MySQL C API
C compiler (e.g., GCC)
